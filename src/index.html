Printable labels 7.3 x 8.3cm

1. Open a spreadsheet like LibreOffice Calc or Excel

Others like Numbers or online tools like Google Sheets should work too.

2. Put in it what you want on the labels (example)

  • 1 column = 1 row on the label.
  • First column is dedicated to the big number/code at the bottom of the label.
  • A column without header is ignored.
  • The style of the spreadsheet doesn't matter so you may go fancy with colors✨!

Here is an example to get started.
3. Ctrl + A then Ctrl + C in the spreadsheet

Or select with the mouse then right click then select Copy

4. Ctrl + V on this page

Or select with the mouse then right click then select Paste
